# Linear Congruential Generator (LCG) - Theory and Implementation

This document explains the mathematical theory behind the Linear Congruential Generator (LCG), one of the oldest and simplest methods for generating pseudo-random numbers. It also summarizes how the provided R code implements this method and analyzes repetitions within the sequence.

---

## 1. Definition of LCG

The Linear Congruential Generator produces a sequence of pseudo-random numbers based on the following recurrence relation:

$$
x_{n+1} = (a \cdot x_n + c) \; \bmod \; m
$$

Where:  
- **a** = multiplier  
- **c** = increment  
- **m** = modulus  
- **x₀** = seed (initial value)  

---

## 2. Properties of LCG

The sequence generated by the LCG is deterministic and eventually enters a cycle because all values are computed modulo *m*. The quality of the generator depends on the choice of parameters (*a, c, m, seed*).

- If parameters are poorly chosen, the sequence will repeat very quickly (**short cycle length**).  
- If parameters are carefully chosen, the LCG can generate a long sequence before repeating.  

---

## 3. Period of LCG

The maximum possible cycle length (period) of an LCG is *m*.  
To achieve the full period, the following conditions must hold (**Hull–Dobell Theorem**):

1. *c* and *m* are relatively prime.  
2. *(a - 1)* is divisible by all prime factors of *m*.  
3. If *m* is a multiple of 4, then *(a - 1)* is also a multiple of 4.  

---

## 4. Example Calculation

Suppose we choose parameters **a = 11, c = 0, m = 32**, and seed **x₀ = 27**.  
The recurrence relation becomes:

$$
x_{n+1} = (11 \cdot x_n) \; \bmod \; 32
$$

This generates a sequence such as: 27, 9, 3, 1, 11, 23, 29, ... until it repeats.
---

## 5. Implementation Summary (R Code)

The R code defines a function **`lcg()`** that:  
- generates the sequence  
- checks for repetition  
- returns both the sequence and the step at which the first repetition occurs  

An auxiliary function **`analyze_sequence()`** computes statistical properties such as:  
- median  
- number of unique values  
- preview of the first numbers
